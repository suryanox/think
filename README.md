---

### Features

- Infinite canvas with pan & zoom
- Hand-drawn style shapes
- Freehand drawing
- Disappearing ink pen
- Dark/Light mode
- Export to JSON
- Keyboard shortcuts

### Quick Start

```bash
yarn install
yarn dev
```

### Shortcuts

| Key | Tool |
|-----|------|
| V | Select |
| R | Rectangle |
| O | Ellipse |
| L | Line |
| A | Arrow |
| P | Pen |
| D | Disappearing Pen |
| T | Text |
